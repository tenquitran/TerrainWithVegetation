TerrainWithVegetation  - a Win32 OpenGL application representing a terrain.

Uses the following libraries:
	GLEW, version 2.0.0 (http://glew.sourceforge.net/)
	GLM, version 0.9.8.4 (http://glm.g-truc.net/)
	SOIL (http://www.lonesock.net/soil.html)

Assets:
	- heightmap: http://terrain.party/
	- textures:  http://seamless-pixels.blogspot.ru/

Created: May 19, 2018
